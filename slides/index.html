What is MicroPython?

MicroPython is a lean and efficient implementation of the Python 3 programming language that includes a small subset of the Python standard library and is optimised to run on microcontrollers and in constrained environments.

Connecting to a REPL

A Read Evaluate Print Loop (REPL) is an excelent way to test your code, and to learn.

Via the REPL you can issue Python commands directly via a serial terminal (or WiFi) to the microcontroller's Python interpreter.

Connection methods vary for different Operating Systems, to learn more follow this REPL Tutorial

For me, I'll be using Linux (ubuntu) and cu 

$ cu -l /dev/ttyACM0

The Basics

File System

The pyboard has the following files stored by default in flash 
/flash/
    README.txt
    boot.py
    main.py
    pybcdc.inf
/sd/
    (sd card's files)
(more about these files later)
The os module can be used to view and manipulate them 
>>> import os
>>> os.listdir('/')
['flash', 'sd']
>>> os.getcwd()
'/flash'
>>> os.listdir('.')
['main.py', 'pybcdc.inf', 'README.txt', 'boot.py']

Hot Mounting SD Card

The SD Card can be mounted without a reset 
>>> import os, pyb
>>> os.umount('/sd')
>>> os.listdir('/')
['flash']
>>> os.mount(pyb.SD, '/sd')
>>> os.listdir('/')
['flash', 'sd']

LEDs

The pyboard has 4 Light Emitting Diodes (LEDs)

  • red
  • green
  • yellow
  • blue 
import pyb

red_led = pyb.LED(1)
green_led = pyb.LED(2)
yellow_led = pyb.LED(3)
blue_led = pyb.LED(4)

# Basic controlls
red_led.on()
red_led.off()
red_led.toggle()
red_led.toggle()

# LED's 3 & 4 have intensity control
yellow_led.intensity(0xff)  # same as on()
yellow_led.intensity(0x00)  # same as off()
